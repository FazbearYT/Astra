## 🎯 О проекте

**Adaptive ML System** — это система, которая автоматически:

1. **Анализирует** ваш датасет (профилирование данных)
2. **Выбирает** оптимальную ML-модель из набора предобученных
3. **Обучает** и **тестирует** выбранную модель
4. **Выдаёт** результаты с метриками качества и визуализациями

### 💡 Основная идея

> Пользователь не должен быть экспертом в ML. Просто загрузите данные → система сама подберёт лучшую модель → получите результат.

## ✨ Возможности

| Функция                       | Описание                                                  |
|-------------------------------|-----------------------------------------------------------|
| 📊 **Автопрофилирование**     | Анализ статистик, распределения классов, сложности данных |
| 🤖 **Автовыбор модели**       | Тестирование 5+ моделей и выбор лучшей по метрикам        |
| 📈 **Визуализация**           | Графики, матрицы ошибок, профили данных                   |
| 💾 **Сохранение результатов** | Модели, метрики, предсказания в удобном формате           |
| 🔍 **Автообнаружение данных** | Программа сама находит датасеты в `data/`                 |
| 🎓 **Встроенные датасеты**    | Iris Dataset доступен без загрузки                        |

### Поддерживаемые модели

- **Random Forest** — для небольших сбалансированных данных
- **SVM (RBF)** — для данных с чёткими границами
- **Gradient Boosting** — для сложных нелинейных паттернов
- **Neural Network (MLP)** — универсальная модель
- **Logistic Regression** — быстрая базовая модель
- **K-Nearest Neighbors** — простая интерпретируемая модель

## 💻 Использование

### Сценарий 1: Первый запуск (без своих данных)

```bash
python app.py
```

**Программа предложит:**

1. ✅ Создать тестовый датасет Iris (150 строк)
2. ✅ Загрузить встроенный Iris Dataset
3. ✅ Положить свой CSV в `data/tabular/`

**Просто выберите опцию 1 или 2 для быстрого старта!**

---

### Сценарий 2: Свой CSV

**1. Подготовьте CSV файл:**

```csv
feature1,feature2,feature3,target
5.1,3.5,1.4,0
4.9,3.0,1.4,0
7.0,3.2,4.7,1
...
```

**Требования к CSV:**

- Кодировка: UTF-8
- Разделитель: запятая
- Минимум 50 строк
- Минимум 2 класса в target
- Числовые признаки (колонки)

**2. Положите файл в папку:**

```
<project_name>/
└── data/
    └── tabular/
        └── my_dataset.csv  ← ваш файл
```

**3. Запустите программу:**

```bash
python app.py
```

**4. Выберите ваш датасет из списка**

---

## ❓ FAQ

### ❓ Какие данные можно загружать?

**Поддерживаются:**

- ✅ CSV файлы с числовыми признаками
- ✅ Минимум 50 строк
- ✅ Минимум 2 класса
- ✅ Кодировка UTF-8

**Не поддерживаются:**

- ❌ Изображения (требует отдельной настройки)
- ❌ Текст без предобработки
- ❌ Данные с пропусками >50%

---

### ❓ Сколько времени занимает обучение?

| Размер данных | Время обучения |
|---------------|----------------|
| 100-500 строк | 10-30 секунд |
| 500-2000 строк | 30-120 секунд |
| 2000-10000 строк | 2-10 минут |

---

### ❓ Где хранятся результаты?

Все результаты в папке `outputs/`:

- `outputs/latest/` → последняя сессия
- `outputs/run_001/` → сессия 1
- `outputs/run_002/` → сессия 2

---

### ❓ Как добавить свою модель?

Откройте `src/model_selector.py` и добавьте в `create_default_models()`:

```python
from sklearn.ensemble import ExtraTreesClassifier

model = SpecializedModel(
    name="ExtraTrees_Specialist",
    model=ExtraTreesClassifier(n_estimators=100),
    profile_requirements={'data_complexity': 'medium'},
    description="Extra Trees Classifier"
)
self.register_model(model)
```

---

### ❓ Ошибка "Нет числовых колонок"

**Причина:** В CSV все колонки текстовые

**Решение:**

1. Проверьте формат CSV (числа без кавычек)
2. Удалите текстовые колонки или закодируйте их
3. Убедитесь что target колонка корректна

---

### ❓ Ошибка при загрузке CSV

**Проверьте:**

- Кодировка файла (должна быть UTF-8)
- Разделитель (запятая, не точка с запятой)
- Нет пустых строк в начале файла
- Заголовки колонок в первой строке
